Linux Commands Guide

Simple Linux Commands

ls - List directory contents
pwd - Show current directory
cd [directory] - Change directory
mkdir [name] - Create a directory
rm [file] - Remove a file
cp [source] [destination] - Copy files

Advanced Linux Commands

grep "text" file - Search for text in a file
find /path -name "filename" - Find files
tar -cvf archive.tar folder/ - Create tar archive
ps aux - Show running processes
kill PID - Kill a process
chmod 755 file - Change file permissions

Hacker Linux Commands

nmap -sV [IP] - Scan open ports
netstat -tulnp - Show network connections
tcpdump -i eth0 - Capture network packets
hydra -l user -P passlist.txt [IP] ssh - Brute force SSH
john --wordlist=rockyou.txt hashfile - Crack password hashes
sqlmap -u "http://target.com/page?id=1" --dbs - SQL injection test
